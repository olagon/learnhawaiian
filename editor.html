How to use this tool
Search or sort to find words. Click any row to fix the Hawaiian or English, change the type or tier, or add a note explaining the fix. Use Mark for deletion on bad entries, and Add entry for new words. Your work auto-saves in this browser. When done, click Export changes and send the downloaded file back. Nothing here touches the live app until those changes are merged.
